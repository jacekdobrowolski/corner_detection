FAST¶

Metoda FAST (Features from accelerated segment test) [Rosten & Drummond, 2006] polega na znalezieniu punktów dla których w danym promieniu. Znaczna część pixeli różni się od centralnego pixela o wartość progu detekcji. Zaletą tej metody jest niska złożoność obliczeniowa. Algorytm ten ma 3 parametry które można zbadać.

corner_detection.FAST(image: numpy.ndarray, radius: int, threshold: int, n: float)¶

Znajduje dobre narożniki w białoczarnym obrazie metodą FAST

Parameters

  • image (np.ndarray) – Białoczarny obraz w postaci 2 wymiarowej np.ndarray

  • radius (int) – Promień pierścienia wokół kandydującego piksela. Musi być równy lub większy niż 1.

  • threshold (int) – Wartość progowa różnicy w jasności.

  • n (int) – Część pikseli z pierścienia, o różnicy wartości jasności z pikselem kandudującym, wymagana do rozpoznania narożnika. Musi być między 0 a 1.

Returns

set Zwraca zbiór par kordynatów znalezionych narożników.

Najpierw importowane są potrzebne moduły

import corner_detection
import cv2
import numpy as np
from matplotlib import pyplot as plt

Działanie dla różnych wartości promienia:¶

img_color = cv2.imread('datasets/Urban2/frame10.png')
img = cv2.cvtColor(img_color, cv2.COLOR_BGR2GRAY)

corners = corner_detection.FAST(image=img, radius=2, threshold=70, n=0.7)

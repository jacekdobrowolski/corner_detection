Optical Flow¶

Lucas-Kanade method¶

optical_flow.lucas_kanade(previous: numpy.ndarray, current: numpy.ndarray, features: set, window_size: int, operator: numpy.ndarray)¶

Implements Lucas Kanade method of calculating optical flow.

Parameters

  • previous (np.ndarray) – previous frame.

  • current (np.ndarray) – current frame.

  • features (set) – Set of features to track.

  • window_size (int) – Size of window around feature, in which optical flow will be calculated.

  • operator (np.ndarray) – Derivative operator like Scharr or Sobel.

Returns

List of tuples containing feature coordinates and calculated transition vector.

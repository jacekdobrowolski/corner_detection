Corner detection methods¶

FAST¶

corner_detection.FAST(image: numpy.ndarray, radius: int, threshold: int, n: float)¶

Finds good corners in grayscale image, using FAST method.

Parameters

  • image (np.ndarray) – Grayscale image in form of 2 dimensional np.ndarray

  • radius (int) – Radius of a ring around candidate corner. Must be greater or equals to 1.

  • threshold (int) – Threshold value.

  • n (int) – Fraction of pixels on ring around candidate corner with absolute value difference greater than threshold. Must be between 0 and 1.

Returns

List of tuples containing indexes of good corners

Harris¶

corner_detection.harris(image: numpy.ndarray, operator: numpy.ndarray, window: numpy.ndarray, k: float)¶

Harris method implementation

Parameters

  • image (np.ndarray) – Image in with corners are to be found.

  • operator (np.ndarray) – Operator used for approximating image derivatives. For example: filter_kernels.SOBEL filter_kernels.SCHARR

  • window (np.ndarray) – Window

  • k (int) – Constant determining algorithm sensitivity, usually beetwen 0.04 and 0.15

Returns

image sized array containing value coresponding to how good of a corner a given pixel is.

Return type

np.ndarray

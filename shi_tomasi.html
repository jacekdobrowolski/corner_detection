Shi-Tomasi¶

Metoda Shi-Tomasi bezpośrednio wyznacza \(min(\lambda_1, \lambda_2)\) Co daje nieco lesze wyniki kosztem złożoności obliczeniowej.

corner_detection.shi_tomasi(image: numpy.ndarray, operator: numpy.ndarray, window: numpy.ndarray, max_corners: int, min_distance: int)¶

Implementuje metodę wykrywania narożników Shi-Tomasi

Parameters

  • image (np.ndarray) – Obraz w tórym nają zostac znalezione narożniki.

  • służący do przybliżenia (Operator) – pochodnych obrazu. Przykładowo: filter_kernels.SOBEL lub filter_kernels.SCHARR

  • window (np.ndarray) – Okno służące do sumowania wartości macierzy struktury.

  • max_corners (int) – Maksymalna liczba narożników jakie zwróci funkcja.

  • min_distance (int) – Minimalny dystans pomiędzy narożnikami jakie zwróci funkcja.

Returns

zwraca zbiór par kordynatów narożników.

Return type

set

Najpierw importowane są potrzebne moduły

import corner_detection
import cv2
import numpy as np
from matplotlib import pyplot as plt

Działanie na przykładowych obrazach:

img_color = cv2.imread('datasets/Urban2/frame10.png')
img = cv2.cvtColor(img_color, cv2.COLOR_BGR2GRAY)

corners = corner_detection.shi_tomasi(img, corner_detection.SCHARR, np.ones((7,7)), 10, 10)

plt.subplot(121)
plt.imshow(img_color)
plt.axis('off');
plt.subplot(122)
plt.imshow(corner_detection.draw_corners(img_color, corners))
plt.axis('off');
